# Opening a black-box


## Get data

In order to get the data go to `black-box/database` and run `get_data.sh`.

## Build cluster

```bash
cd spark-config
./build_cluster.sh
```

## Run cluster locally

```bash
cd spark-config
./run_cluster_locally.sh
```

## Submit black-box

```bash
cd black-box
./submit.sh
```
